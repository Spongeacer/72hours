# 72Hours 事件库 (Events Library)

> 历史锚点与非关键事件
> 基于物理机制触发，涌现式呈现

---

## 第一部分：关键历史锚点 (Historical Anchors)

关键锚点是历史大势的节点，不可改变，但形态受物理状态影响。

---

### 锚点1：暗流涌动 (The Undercurrent)

**回合**：第1回合（1851年1月8日凌晨）
**性质**：固定开场
**触发**：游戏开始自动触发

**物理状态**：
- P_history = 10（初始压强）
- Ω = 1.0（初始全局因子）
- 天气：夜

**涌现描述**：
```
玩家醒来，发现世界已经不一样了。

可能涌现：
- 远处有火光，不是灯笼（会众集结）
- 母亲/关联NPC在门外徘徊（担忧）
- 窗外有脚步声，不是一个人的（官兵巡逻）
- 什么都没有，但空气凝重（压抑）

叙事示例：
> 你被一阵奇怪的声音惊醒。
> 不是鸡鸣，是人在低语，很多声音叠在一起，像潮水。
> 你走到窗边，看到远处有火光，不是灯笼的颜色。
> 这是金田村，1851年1月8日，凌晨。
> 你不知道，但历史已经开始了。
```

**效果**：
- 生成2个关联NPC（基于玩家身份）
- 随机分布普通NPC
- 设置初始天气、P、Ω

### 锚点2：官兵搜查 (The Midnight Raid)

**回合**：第24回合±6（1851年1月9日深夜）
**性质**：动态锚点

**提前触发条件**：
- P > 50（局势紧张）
- 或 玩家戾气值 > 70（暴力信号广播）
- 或 玩家被举报（关联NPC出卖）

**延后触发条件**：
- P < 30（表面平静）
- 且 玩家质量 M < 4（无威胁）

**物理状态**：
- P_jump = +25（压强跃升）
- 信号：[宵禁]、[搜捕]
- 刷新3个[官兵]NPC于玩家附近

**涌现描述**：
```
官兵不是预设剧情，是P和F达到阈值后的自然涌现。

可能涌现：
- 破门而入（冲突行为）
- 窗外窥视（偷听行为）
- 敲门询问（请求行为）
- 径直走过（偶然）

叙事示例：
> 门被撞开的时候，你正在...
> （基于玩家当前行为生成）
> 
> 三个官兵站在门口，雨水顺着他们的刀尖滴落。
> 领头的那个看着你，眼神里有...
> （基于官兵特质生成：贪婪/嗜血/疲惫/恐惧）
> 
> "搜。"他说。
```

**效果**：
- P_base + 25
- 解锁清军营官精英NPC（若未解锁）
- [教徒]身份玩家面临暴露风险
- 关联NPC可能在此事件中被抓/死亡

### 锚点3：天父下凡 (The Divine Possession)

**回合**：第48回合±4（1851年1月10日黄昏）
**性质**：动态锚点

**提前触发条件**：
- 会众NPC质量总和 > 阈值（集体觉醒）
- 或 杨秀清对玩家引力 F > 8（核心凝聚）
- 或 杨秀清已解锁且 K > 5

**延后触发条件**：
- 会众分散（质量分散）
- 或 杨秀清质量 M < 6（领袖失能）

**物理状态**：
- Ω 显著增长（中期跃升）
- 所有[教徒]NPC质量临时+2
- 信号：[狂热]、[集结]

**涌现描述**：
```
杨秀清以"天父下凡"形式发布动员。
这不是预设剧情，是会众质量达到临界后的集体行为。

可能涌现：
- 玩家在场，目睹全过程（被卷入）
- 玩家不在场，事后听说（外部视角）
- 玩家试图阻止（冲突）
- 玩家试图加入（被接纳或质疑）

叙事示例：
> 黄昏，紫荆山下。
> 
> 杨秀清站在石头上，突然浑身颤抖。
> 他的声音变了，变得苍老、威严，不像他。
> "我乃天父，下凡诛妖。"
> 
> 人群跪下，像潮水一样。
> 你...
> （基于玩家身份和状态生成选择）
```

**效果**：
- 所有[教徒]NPC向原点移动
- 玩家被迫选择立场（支持/质疑/沉默/逃离）
- 解锁韦昌辉（若玩家有财富）
- 起义进度大幅推进

### 锚点4：万寿祝寿 (The Birthday Vigil)

**回合**：第71回合（1851年1月11日子夜）
**性质**：固定前奏

**物理状态**：
- Ω 达到临界值（末期跃升）
- 所有已解锁精英NPC强制出现在玩家附近
- P 达到峰值
- 质量坍缩即将发生

**涌现描述**：
```
洪秀全38岁生日前夜。
所有命运线强制交织，无论玩家如何选择。

可能涌现：
- 玩家与洪秀全直接对话（若在场）
- 玩家被排斥在核心之外（若不在场）
- 玩家试图阻止起义（徒劳）
- 玩家试图加入核心（被接纳或拒绝）

叙事示例：
> 子夜，金田村。
> 
> 你从未见过这么多人聚集在一起。
> 火光把夜空烧红，像末日，也像诞生。
> 
> 洪秀全站在中央，黄袍在火光中像一面旗帜。
> 他的目光扫过人群，扫过你。
> 
> "明日，"他说，"天国降临。"
> 
> 这不是预言，是宣告。
> 历史已经不可阻挡。
```

**效果**：
- 所有精英NPC（已解锁）强制聚集
- 玩家质量M若足够高，成为核心人物
- 玩家质量M若低，被裹挟或边缘化
- 最终选择机会（逃离/加入/观望/对抗）

### 锚点5：金田起义 (The Uprising)

**回合**：第72回合（1851年1月11日凌晨）
**性质**：固定结局，不可改变

**物理状态**：
- Ω 达到最大值
- 全局质量坍缩
- 所有NPC向原点疯狂撞击
- 起义爆发

**涌现描述**：
```
历史大势，不可阻挡。
玩家的选择只决定自己在这场大势中的位置。

可能涌现：
- 玩家在起义队伍中（主动加入）
- 玩家在远处观望（被动卷入）
- 玩家在逃亡路上（逃离）
- 玩家在狱中/死亡（对抗失败）

叙事示例：
> 凌晨，金田村。
> 
> 第一声号角响起的时候，你...
> （基于玩家最终选择生成）
> 
> 红巾裹头，刀枪出鞘。
> 人群像潮水一样涌出村庄，向未知涌去。
> 
> 这是1851年1月11日，太平天国元年。
> 你在这里，或不在这里。
> 历史记住了这一天，
> 是否记住你，取决于你留下了什么。
```

**效果**：
- 游戏进入结局结算
- 生成2000字人物小传
- 记录玩家遇到的所有故事
- 显示"如果活着会看到什么"
- 显示玩家对历史的影响（微小但存在）

## 第二部分：非关键事件 (Minor Events)

非关键事件随机触发，增加世界丰富度，不影响历史大势。

---

### 事件1：外国传教士进村

**触发条件**：
- 随机，低概率
- 回合 10-50之间
- P < 40（相对平静时）

**涌现描述**：
```
一个金发碧眼的外国人，背着十字架，说着蹩脚的中文。

可能涌现：
- 玩家是[教徒]：被误认为是同伙，或被质疑信仰纯正
- 玩家是[读书人]：尝试用英语交流，获取外部世界信息
- 玩家是[官兵]：怀疑是间谍，试图抓捕
- 玩家是[地主]：好奇，试图交易

叙事示例：
> 他在村口被一群孩子围着，像看猴子一样。
> 看到你，他眼睛一亮，用蹩脚的中文说："平安？"
> 他的十字架和你的不一样，更大，更复杂。
> 
> 你意识到，这个世界比金田村大得多。
```

**效果**：
- 短暂改变P（好奇或紧张）
- 可能获得[圣经]/[外部消息]道具
- 可能引发村民议论

---

### 事件2：官府征粮

**触发条件**：
- 随机，中等概率
- 回合 15-40之间
- 玩家身份为[地主]/[农民]

**涌现描述**：
```
官兵带着秤和袋子，挨家挨户收粮。

可能涌现：
- 玩家是[地主]：被要求多交，可能贿赂或反抗
- 玩家是[农民]：被迫交出救命粮，可能逃亡或反抗
- 玩家是[官兵]：被要求协助，面临道德选择
- 其他身份：旁观，或被波及

叙事示例：
> 官兵的秤有问题，谁都知道。
> 但知道又能怎样？
> 
> 老张站在旁边，看着自己一年的收成被装走。
> 他的眼神...
> （基于玩家与老张的关系生成）
```

**效果**：
- P +10（民怨增加）
- [农民]NPC恐惧/怨恨增加
- 可能触发抗粮冲突
- 为后续起义积累情绪

---

### 事件3：瘟疫蔓延

**触发条件**：
- 随机，低概率
- 回合 20-50之间
- 天气为[雨]/[雾]

**涌现描述**：
```
村里有人发烧，然后死去。恐慌蔓延。

可能涌现：
- 玩家尝试救助（需要[郎中]或药物）
- 玩家逃离疫区
- 玩家被感染（状态[患病]）
- 关联NPC被感染（情感冲击）

叙事示例：
> 第一具尸体被抬走的时候，还没有人在意。
> 第三具的时候，村里开始传言是"天罚"。
> 
> 你看着那些发烧的人，他们的眼睛...
> 不像人，像被什么东西占据了。
```

**效果**：
- 部分NPC随机消失（死亡）
- P +15（恐慌）
- 玩家可能获得[患病]状态
- [郎中]NPC价值暴增

---

### 事件4：流民涌入

**触发条件**：
- 随机，中等概率
- 回合 10-60之间
- P > 40（局势紧张时概率增加）

**涌现描述**：
```
从湖南、广东逃来的流民，带着饥饿和恐惧。

可能涌现：
- 玩家给予帮助（K增加，但资源消耗）
- 玩家拒绝帮助（道德压力）
- 流民中有[教徒]，带来外部消息
- 流民中有[官兵]眼线，收集情报

叙事示例：
> 他们像潮水一样涌进村子，又脏又饿。
> 有人怀里抱着已经死去的孩子，不肯放手。
> 
> 你给他们一碗粥，他们跪下磕头。
> 你不给他们，他们就用那种眼神看着你...
> 像看一个罪人。
```

**效果**：
- 刷新多个[流民]NPC
- P +5（资源紧张）
- 可能获得外部情报
- 可能触发抢夺/冲突

---

### 事件5：乡绅宴会

**触发条件**：
- 随机，低概率
- 回合 10-40之间
- 玩家身份为[地主]/[读书人]
- P < 30（相对平静）

**涌现描述**：
```
本地乡绅王作新举办宴会，玩家被邀请或旁观。

可能涌现：
- 玩家参加：获取上层信息，但可能被要求表态
- 玩家拒绝：被视为异类
- 玩家在窗外偷听：获取秘密情报
- 宴会被会众袭击：突然卷入冲突

叙事示例：
> 灯火通明，酒肉飘香。
> 与村外的饥饿形成对比。
> 
> 王作新举杯，说："剿匪，保境安民。"
> 众人附和，但你注意到...
> 有人没有举杯，眼神闪烁。
```

**效果**：
- 获取官府/乡绅动向情报
- 可能被迫表态（支持/反对会众）
- 可能触发[告密]事件
- 关联NPC（如教书先生）可能在场

### 事件6：天象异变

**触发条件**：
- 随机，极低概率
- 任意回合
- 夜晚

**涌现描述**：
```
血月、彗星、日食...古人眼中的天兆。

可能涌现：
- [教徒]：视为上帝征兆，狂热增加
- [读书人]：试图用知识解释，但内心不安
- [官兵]：视为不祥，恐惧增加
- 所有人：P波动（迷信影响）

叙事示例：
> 月亮是红色的，像浸在血里。
> 
> 有人说这是"天诛清妖"的征兆。
> 有人说这是"大乱将至"的警告。
> 
> 你看着那轮红月，不知道它预示着什么。
> 但你感觉到，有什么东西正在改变。
```

**效果**：
- P 随机波动（±20）
- [狂热]NPC质量临时增加
- [怯懦]NPC恐惧增加
- 可能触发[疯癫]状态

## 事件触发总览

| 事件类型 | 数量 | 触发方式 | 对历史的影响 |
|----------|------|----------|--------------|
| 关键锚点 | 5 | 回合/物理条件 | 决定历史走向 |
| 非关键事件 | 6+ | 随机/条件 | 丰富世界，不影响大势 |

---

## 事件与涌现

所有事件，无论关键与否，都遵循涌现原则：
- 不是"触发剧情"
- 而是"物理状态达到临界后的自然现象"
- 玩家的存在改变事件的形态
- 但不改变事件的发生（关键锚点）
