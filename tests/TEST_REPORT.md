## 4. 完整故事线示例（读书人身份）

### 开场

```
> 你被一阵奇怪的声音惊醒。
> 不是鸡鸣，是人在低语，很多声音叠在一起，像潮水。
> 你走到窗边，看到远处有火光，不是灯笼的颜色。
> 这是金田村，1851年1月8日，凌晨。
> 你不知道，但历史已经开始了。
```

**初始状态**：
- 身份：村中的读书人
- 基础质量 B：3
- 特质：[读书人]、[冷静]、[好奇]
- 执念：[发现隐藏的真相]
- 关联NPC：母亲(K=5)、教书先生(K=2)

---

### 第1回合（1月8日 01:00，night）

**系统状态**：
- 压强 P：10.5
- 全局因子 Ω：1.02
- 天气：night

**聚光灯**：教书先生

**叙事**：
```
1851年1月8日 01:00，night。你遇到了教书先生。空气中弥漫着紧张的气息。
```

**选择**：
1. 你沉默地看着对方，手按在刀柄上
2. 你开口说话，声音比想象中更沙哑
3. 你转身，假装没有看见

**玩家选择**：选项2（开口说话）

**结果**：教书先生透露最近有人在查读书人，P -2（缓冲型特质）

---

### 第2回合（1月8日 02:00，night）

**系统状态**：
- 压强 P：11.0
- 全局因子 Ω：1.04
- 天气：night

**聚光灯**：母亲

**叙事**：
```
母亲站在门口，手里端着一碗热粥。
"外面乱，喝完这碗，别出门了。"
她的眼睛里有担忧，但更多的是你不知道的东西。
```

**选择**：
1. 接过粥，默默喝完
2. 告诉她你要出去看看
3. 把粥放在一边，继续收拾行囊

**玩家选择**：选项1（接过粥）

**结果**：K +0.5，母亲质量轻微增加（引力陷阱启动）

---

### 第3-10回合（1月8日白天）

**关键事件**：
- 第3回合：遇到逃兵李三，给予食物，K +1
- 第5回合：天气转为clear，P增长速度减缓
- 第7回合：发现官兵巡逻，P +5
- 第9回合：与教书先生深度交谈，获得[线装书]道具

**系统状态（第10回合）**：
- 压强 P：25
- 全局因子 Ω：1.20
- 已解锁NPC：无（精英NPC条件未满足）

---

### 第20回合（1月8日 20:00，night）

**系统状态**：
- 压强 P：35
- 全局因子 Ω：1.40

**聚光灯**：逃兵李三（已升格，K=5）

**叙事**：
```
李三站在阴影里，手里握着一把刀。
"你救过我，"他说，"但现在我需要更多。"
他的眼神变了，不再是那个逃兵，而是某种更危险的东西。
```

**选择**：
1. 给他银两，让他离开
2. 拒绝，但保持警惕
3. 试图说服他加入你

---

### 第24回合（1月9日 00:00）- 锚点2：官兵搜查

**触发条件**：P=38 < 50，正常触发

**系统状态**：
- 压强 P：40（事件+10）
- 刷新3个官兵NPC
- 聚光灯：清军营官

**叙事**：
```
门被撞开。三个官兵站在门口，雨水顺着刀尖滴落。
领头的营官看着你，眼神里有贪婪，也有疲惫。
"搜。"他说。

你怀里藏着那本线装书。他的目光扫过你的胸口，停住了。
你知道他看见了。他也知道你知道。

空气凝固。
```

**选择**：
1. 主动交出书籍，请求宽恕
2. 沉默，等待他做决定
3. 拔腿逃跑

**结果**：选择2，营官犹豫后离开，但记住你的脸（hostility +20）

---

### 第40回合（1月9日 16:00）- 冯云山解锁

**解锁条件**：回合≥40，满足

**系统状态**：
- 冯云山出现在坐标(2,2)
- 质量 B=7，加上S/K后总质量≈10
- 对玩家引力 F=15（高）

**叙事**：
```
一个面色苍白的教书先生坐在油灯下。
他抬头看你，目光温和却疲惫："你也来寻道的吗？"

你注意到他手中的册子，封面上有你不认识的符号。
那是上帝的文字，你想。
```

---

### 第48回合（1月10日 00:00）- 锚点3：天父下凡

**触发条件**：会众质量总和=45 < 60，延后触发
- 实际触发：第50回合

**系统状态**：
- 杨秀清解锁（第50回合）
- 会众NPC质量总和=65 > 60
- 触发天父下凡

**叙事**：
```
黄昏，紫荆山下。

杨秀清站在石头上，突然浑身颤抖。
他的声音变了，变得苍老、威严，不像他。
"我乃天父，下凡诛妖。"

人群跪下，像潮水一样。
你...
```

**选择**：
1. 跪下（获得[教徒]标签）
2. 站着，但低头
3. 转身离开

### 第60回合（1月10日 12:00）- 洪秀全解锁

**解锁条件**：回合≥60，满足

**系统状态**：
- 洪秀全 B=10，总质量≈15
- Ω=2.0（指数增长开始）
- 所有NPC向原点移动

**叙事**：
```
远处传来低沉的诵经声。一个身穿黄袍的身影在火光中显现。
"我乃上帝次子，下凡诛妖。"

他的目光扫过你，"你，可是来投天国的？"
```
