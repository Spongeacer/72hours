# 72Hours 涌现式叙事机制总览

## 一、核心涌现原则

### 1.1 涌现式叙事定义
```
输入：此刻的所有在场者 + 环境 + 信号 + 记忆
↓
共鸣（不是计算，是共振）
↓
故事自然流淌
```

### 1.2 与预设叙事的区别
| 预设叙事 | 涌现式叙事 |
|---------|-----------|
| if-else 条件判断 | 感觉对感觉的碰撞 |
| 固定文本模板 | 基于情境的独特生成 |
| 玩家控制故事 | 玩家是催化剂，故事自己长 |
| 确定性结果 | 随机扰动 + 无因之果 |

## 二、物理涌现机制

### 2.1 引力系统
```
F = G × M₁ × M₂ / r² × (1 + Ω × pressureMultiplier)
```

| 参数 | 说明 | 涌现效果 |
|-----|------|---------|
| G | 引力常数 (0.8) | 基础吸引力 |
| M | 质量 = B + S + K + O | 质量越大，引力越强 |
| r | 距离 | 距离越远，引力越弱 |
| Ω | 全局因子 | 历史必然性增强引力 |
| P | 压强 | 环境紧张度放大引力 |

**涌现现象**：
- **引力捕获**：高S/K路人被拉入核心圈
- **双星系统**：两高M NPC互吸，玩家被甩出
- **质量坍缩**：NPC死亡→M变负→心理创伤

### 2.2 质量模型
```
M = B (基础) + S (叙事) + K (关系) + O (道具)
```

| 分量 | 获取方式 | 涌现特性 |
|-----|---------|---------|
| B | 角色预设 | 先天位格，不变 |
| S | 经历事件累积 | 活得越久，质量越高 |
| K | 交互次数×0.5 | 纠缠越深，质量越大 |
| O | 持有关键道具 | 瞬间暴增，可流转 |

### 2.3 压强系统
```
P = P_history + violenceEvents × 2
```

**涌现效果**：
- P > 50：空气中弥漫不安
- P > 70：烟味和血腥味
- P > 90：命运感笼罩所有人

### 2.4 Ω 全局因子
```
if (pressure >= 60) Ω = Ω × 1.05  // 指数增长
else Ω += 0.02  // 线性增长
```

**涌现效果**：
- Ω > 2：历史必然性显现
- Ω > 3：无法逃避的命运感
- Ω > 4：所有人撞向原点，命运交织

### 2.5 NPC 移动涌现
```
移动 = 引力方向 × 恐惧修正 × K值修正 + 随机漂移
```

| 条件 | 修正 | 涌现行为 |
|-----|------|---------|
| fear > 70 | 反向 × 2 | 恐慌逃跑 |
| knot > 5 | × 1.5 | 紧紧跟随 |
| knot < 0 | 反向 | 主动远离 |
| 随机 | ±范围 | 无因之果 |

## 三、叙事涌现机制

### 3.1 聚光灯选择器
```
聚光灯NPC = max(引力 × 随机扰动[0.8-1.2])
```

**涌现特性**：
- 80%引力决定 + 20%随机
- 不是最强的一定被选中
- 无因之果：有时路人突然成为焦点

### 3.2 行为涌现器
```
感觉场 = [恐惧, 攻击性, 饥饿, 依恋, 吸引, 特质驱动...]
行为 = 最强感觉 × 随机选择(前3)
```

**六类行为涌现**：

| 行为 | 驱动感觉 | 社会学/心理学机制 | 涌现场景示例 |
|-----|---------|------------------|-------------|
| **抢夺** | 贪婪/恐慌 | **社会交换理论**（Homans）：资源稀缺下的成本-收益计算；**恐惧管理理论**（Greenberg）：死亡焦虑导致自我保存优先 | "他的手伸向你的腰间，眼神却在躲闪" |
| **冲突** | 嗜血/仇恨 | **挫折-攻击假说**（Dollard）：目标受阻引发攻击；**社会认同理论**（Tajfel）：内群体偏爱与外群体敌意 | "刀光闪过，你感到胸口一凉" |
| **偷听** | 好奇/狡诈 | **信息缺口理论**（Loewenstein）：未知信息引发认知不适；**博弈论**：信息不对称下的策略优势 | "他躲在门后，呼吸声比雨声还轻" |
| **聊天** | 孤独/共鸣 | **依恋理论**（Bowlby）：安全基地寻求；**社会渗透理论**（Altman）：自我披露增进亲密 | "两个读书人，在乱世油灯下谈诗" |
| **请求** | 绝望/信任 | **习得性无助**（Seligman）：反复失败后的被动依赖；**信任博弈**：脆弱性展示换取合作 | "他跪下，不是因为软弱，是别无选择" |
| **给予** | 慈悲/报恩 | **互惠规范**（Gouldner）：社会交换的普遍道德；**共情-利他假说**（Batson**：情绪共鸣驱动助人 | "她把最后一块干粮塞给你，像母亲" |

#### 3.2.1 社会学视角补充

**1. 社会角色理论（Goffman）**
```
每个NPC都在"表演"其社会角色：
- 母亲：保护者角色 → 给予行为概率+30%
- 教书先生：知识权威角色 → 聊天/偷听行为概率+20%
- 士兵：暴力垄断角色 → 冲突行为概率+40%
```

**2. 社会资本理论（Putnam）**
```
K值 = 社会资本（信任+互惠网络）
- 高K值：强关系 → 给予/请求行为
- 低K值：弱关系 → 偷听/抢夺行为
- 负K值：敌对关系 → 冲突行为
```

**3. 失范理论（Durkheim）**
```
压强P = 社会失范程度
- P↑ → 规范瓦解 → 抢夺/冲突行为↑
- P>70 → "失范状态" → 所有行为趋于极端
```

#### 3.2.2 心理学视角补充

**1. 马斯洛需求层次**
```
行为优先级（从低到高）：
抢夺/觅食 → 安全需求（P↑时优先）
冲突 → 安全/尊重需求
偷听 → 认知需求
聊天 → 归属需求
请求 → 归属/安全需求
给予 → 自我实现需求（高K值时）
```

**2. 认知评价理论（Lazarus）**
```
NPC对情境的初级评价 → 情绪唤醒 → 行为选择
- 威胁评价 → 恐惧 → 逃跑/请求
- 挑战评价 → 攻击性 → 冲突
- 无关评价 → 好奇 → 偷听/聊天
```

**3. 心理距离理论（Trope）**
```
距离影响行为抽象程度：
- 距离近（r<2）：具体、情绪化行为（抢夺/给予）
- 距离远（r>5）：抽象、策略化行为（偷听/观望）
```

#### 3.2.3 涌现的临界点

| 临界条件 | 社会学解释 | 心理学解释 | 涌现行为 |
|---------|-----------|-----------|---------|
| P>50, fear>60 | 社会规范瓦解 | 集体恐慌 | 恐慌性抢夺 |
| knot>5, fear<30 | 强关系信任 | 安全依恋 | 无条件给予 |
| aggression>70, P>60 | 暴力合法化 | 去抑制 | 极端冲突 |
| hunger>70, knot<-3 | 社会契约破裂 | 生存本能 | 背叛性抢夺 |

### 3.3 环境信号系统
```
信号 = f(天气, 压强, Ω, 时间)
```

| 信号类型 | 触发条件 | 情绪效果 |
|---------|---------|---------|
| 雨声 | weather=rain | 忧郁、掩盖 |
| 浓雾 | weather=fog | 神秘、看不清 |
| 夜色 | weather=night | 压抑、隐藏 |
| 不安 | P>50 | 焦虑、紧张 |
| 血腥 | P>70 | 恐惧、危险 |
| 宿命 | Ω>3 | 无法逃避 |
| 紧迫 | turn>60 | 时间无多 |

### 3.4 集体情绪计算
```
集体情绪 = avg(所有在场者 fear, aggression)
```

| 情绪状态 | 条件 | 叙事影响 |
|---------|------|---------|
| 恐慌蔓延 | P>70 && avgFear>60 | 混乱、失控 |
| 暴力酝酿 | avgAggression>60 | 冲突即将爆发 |
| 恐惧笼罩 | avgFear>60 | 压抑、不安 |
| 紧张不安 | P>50 | 等待、猜测 |
| 沉默等待 | 其他 | 平静、积累 |

### 3.5 玩家气场
```
气场 = f(状态, 特质)
```

| 状态 | 特质 | 气场 | 影响 |
|-----|------|-----|------|
| fear>70 | - | 恐惧的颤抖 | 周围NPC fear+ |
| aggression>70 | - | 压抑的愤怒 | 周围NPC aggression+ |
| hunger>70 | - | 饥饿的虚弱 | 可能引发抢夺 |
| injury>50 | - | 带伤的疲惫 | 可能引发慈悲 |
| - | calm | 沉默的警惕 | 稳定周围 |
| - | curious | 探究的目光 | 引发对话 |
| - | brave | 坚定的姿态 | 鼓舞周围 |

## 四、选择涌现机制

### 4.1 NPC 出现概率
```
出现概率 = 距离因子 × 0.6 + K值因子 × 0.4
距离因子 = max(0, 1 - distance/5)
K值因子 = (knot + 10) / 20
```

**涌现效果**：
- 距离近 + 关系好 = 必然出现
- 距离远 + 关系好 = 隐藏选择
- 距离近 + 关系差 = 可能出现（紧张）

### 4.2 选择类型涌现

| 选择 | 涌现条件 | 类型 |
|-----|---------|------|
| 探索 | 始终 | normal |
| 休息 | 始终 | normal |
| 观察 | 始终 | normal |
| 与NPC交谈 | 出现概率 > 0.3 | normal/hidden |
| 逃跑 | fear > 60 | hidden |
| 觅食 | hunger > 60 | normal |
| 暴力 | aggression > 70 | hidden |

### 4.3 结果涌现（非写死）

**探索**：
```
发现 = f(压强, Ω, 随机)
P>50: 远处有火光
Ω>2: 空气紧张
随机: 刀/脚印/口音/乌鸦
```

**休息**：
```
休息质量 = max(0, 100 - P - fear)
>70: 睡得很沉
>40: 断断续续
<40: 无法入睡
```

**交谈**：
```
对话内容 = f(NPC状态, K值, 记忆)
fear>60: "我怕"
hunger>60: 眼神饥饿
knot>5: 握手沉默
其他: 随机话题
```

## 五、记忆涌现机制

### 5.1 记忆影响行为
```
行为倾向 += 记忆强度 × 情感权重
```

| 记忆类型 | 影响 | 涌现效果 |
|---------|------|---------|
| 背叛记忆 | 信任- | 防备、敌意 |
| 感恩记忆 | 信任+ | 帮助、给予 |
| 共同恐惧 | 团结+ | 互相依靠 |
| 冲突记忆 | 敌意+ | 报复、回避 |

### 5.2 记忆触发叙事
```
如果 spotlightNPC.记忆 相关于 当前情境:
    叙事中暗示记忆
```

示例：
- NPC曾被玩家救助 + 当前危险 → "他看你的眼神，像看救命稻草"
- NPC曾与玩家冲突 + 当前紧张 → "他的手握着刀柄，想起上次的事"

## 六、随机扰动（无因之果）

### 6.1 扰动注入点

| 机制 | 扰动范围 | 效果 |
|-----|---------|------|
| 聚光灯选择 | 引力 × [0.8, 1.2] | 第二重要NPC可能上位 |
| 行为选择 | 前3感觉随机选 | 不是最强一定发生 |
| 探索发现 | 50%随机 | 有时一无所获 |
| 交谈话题 | 随机选择 | 对话不可预测 |
| 移动漂移 | ±随机范围 | 位置不完全确定 |

### 6.2 涌现的不可预测性
```
确定性 = 80%系统 + 20%混沌
```

**设计意图**：
- 让玩家感觉世界"活着"
- 每次游戏体验不同
- 无法完全掌控，只能影响

## 七、AI 共振生成（当接入AI时）

### 7.1 共振提示词结构
```
【此刻的所有在场者】
- 玩家：身份、特质、执念、气场、状态
- 聚光灯NPC：执念、特质、状态、行为、记忆

【环境信号】
- 天气、压强、Ω、集体情绪

【共鸣要求】
- 不解释，直接呈现
- 让情绪自然流露
- 暗示而非说明
- 留白给想象
```

### 7.2 离线模式涌现
当无AI时，使用模板组合：
```
叙事 = 集体情绪 + 最强信号 + 聚光灯行为 + 玩家气场
```

## 九、关键涌现公式汇总

| 机制 | 公式 | 说明 |
|-----|------|------|
| 引力 | F = G×M₁×M₂/r²×(1+Ω×P) | 核心物理力 |
| 质量 | M = B + S + K + O | 叙事权重 |
| 压强 | P += 0.8 + violence×2 | 环境紧张度 |
| Ω | Ω×1.05 (if P≥60) | 历史必然性 |
| 出现概率 | dist×0.6 + knot×0.4 | NPC可见性 |
| 聚光灯 | max(force × random[0.8,1.2]) | 焦点选择 |
| 行为 | max(感觉场) × 随机(前3) | 行动涌现 |
| 休息质量 | 100 - P - fear | 恢复效果 |

## 十、涌现 vs 传统实现对比

| 方面 | 传统实现 | 涌现实现 |
|-----|---------|---------|
| 叙事 | 4条固定文本随机 | 基于情境动态生成 |
| NPC出现 | 解锁即出现 | 距离+K值+随机 |
| 行为 | switch-case硬编码 | 感觉场涌现 |
| 结果 | 固定数值变化 | 情境依赖计算 |
| 移动 | 固定方向 | 引力+恐惧+K+随机 |
| 记忆 | 存储无影响 | 影响行为和叙事 |
| 可预测性 | 高 | 中（80%系统+20%混沌）|
| 重玩价值 | 低 | 高 |
