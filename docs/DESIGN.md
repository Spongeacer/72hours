## 1. 核心设计理念

### 1.1 玩家作为催化剂

玩家不是故事的主角，而是**催化剂**——在场，影响，但故事自己流淌。

```
无玩家时：故事默默进行，无人知晓
有玩家时：玩家的存在改变周围的"场"，故事因你而变

玩家可能：
- 不说话，只是也在（陪伴催化）
- 给食物（恩情催化）
- 讲述恐惧（共鸣催化）
- 沉默磨刀（紧张催化）

故事因你而变，但不是由你决定。
```

### 1.2 涌现式叙事

**不是预设公式，而是故事自己长出来。**

```
输入：此刻的所有在场者 + 环境 + 信号 + 记忆
↓
共鸣（不是计算，是共振）
↓
故事自然流淌
```

**涌现的源头**：
- NPC的执念（此刻什么对他最重要）
- NPC的记忆（你让他想起谁）
- 环境信号（雨、血、远处的喊杀）
- 玩家的"气场"（你散发出什么）
- 纯粹的偶然（随机扰动，无因之果）

## 3. 物理引擎

### 3.1 引力公式

```
F = G × M₁ × M₂ / r² × P × Ω
```

| 符号 | 含义 | 默认值 | 范围 |
|------|------|--------|------|
| G | 引力常数 | 0.8 | - |
| M₁, M₂ | 双方质量 | - | 1-20 |
| r | 欧几里得距离 | - | ≥0.1 |
| P | 环境压强 | 1 + pressure × 0.05 | 1-2 |
| Ω | 全局因子 | - | 1-20 |

### 3.2 质量模型

```
M = B + S + K + O
```

| 分量 | 全称 | 说明 |
|------|------|------|
| B | Base（基础质量） | 角色预设 |
| S | Story（叙事权重） | 经历事件累积 |
| K | Knot（关系系数） | 交互次数 |
| O | Object（道具加成） | 持有道具 |

### 3.3 压强与Ω系统

**压强 (Pressure)**: 1-20
- 初始值: 2
- 每回合增长: 0.16
- 表示环境紧张程度

**全局因子 (Ω)**: 1-20
- 初始值: 2
- 基础增长: 0.4/回合
- 高压时(≥12)加速: ×1.02
- 表示历史必然感

### 3.4 物理叙事现象

| 现象 | 机制 | 叙事效果 |
|------|------|----------|
| **引力捕获** | 路人S/K飙升 | 路人变生死之交 |
| **质量坍缩** | NPC死亡→M变负 | 心理创伤 |
| **双星系统** | 两高M NPC互吸 | 玩家被甩出 |
| **全局坍缩** | Ω指数增长 | 命运交织 |

## 4. 行为模式库

### 4.1 六类行为

| 行为 | 核心驱动力 | 可能涌现的场景 |
|------|------------|----------------|
| 抢夺 | 贪婪/恐慌 | "他的手伸向你的腰间，眼神却在躲闪" |
| 冲突 | 嗜血/仇恨 | "刀光闪过，你感到胸口一凉" |
| 偷听 | 好奇/狡诈 | "他躲在门后，呼吸声比雨声还轻" |
| 聊天 | 孤独/共鸣 | "两个读书人，在乱世的油灯下谈起诗" |
| 请求 | 绝望/信任 | "他跪下，不是因为软弱，是因为别无选择" |
| 给予 | 慈悲/报恩 | "她把最后一块干粮塞给你，像母亲一样" |
