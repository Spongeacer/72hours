> 基于王国风云(CK3)风格设计的完整特质系统

## 1. 性格特质 (Personality Traits)

性格特质是角色最核心的属性，影响行为倾向和压力反应。
角色最多拥有3个性格特质，部分特质互斥。

### 1.1 胆量维度 (Boldness)

| 特质ID | 名称 | 类型 | 对P的影响 | 行为倾向 | 互斥 |
|--------|------|------|-----------|----------|------|
| brave | 勇敢 | 激化型 | P×1.1 | 冲突+, 恐惧增长-, 死亡风险+ | craven |
| craven | 怯懦 | 缓冲型 | P×0.9 | 冲突-, 逃跑+, 恐惧增长+ | brave |

**勇敢 (Brave)**:
- 冲突倾向 +0.3
- 恐惧值增长 -20%
- 面对危险时保持镇定
- 容易成为他人依靠的对象

**怯懦 (Craven)**:
- 冲突倾向 -0.3
- 逃跑/请求倾向 +0.3
- 恐惧值增长 +20%
- 高压下容易崩溃

### 1.2 情绪维度 (Temperament)

| 特质ID | 名称 | 类型 | 对P的影响 | 行为倾向 | 互斥 |
|--------|------|------|-----------|----------|------|
| calm | 冷静 | 缓冲型 | P×0.8 | 理智选择权重+, 压力恢复+ | wrathful |
| wrathful | 暴怒 | 激化型 | P×1.3 | 冲突+, 暴力选项解锁 | calm |

**冷静 (Calm)**:
- 高压下保持理智判断
- 压力恢复速度 +50%
- 不易被激怒

**暴怒 (Wrathful)**:
- 冲突倾向 +0.4
- 暴力选项权重增加
- 容易因小事升级冲突
- 威慑力 +20

### 1.3 欲望维度 (Desire)

|--------|------|------|-----------|----------|------|
| chaste | 节制 | 缓冲型 | P×0.9 | 欲望相关事件免疫 | lustful |
| lustful | 色欲 | 中性 | 无 | 勾引/被勾引+, 生育力+ | chaste |
| content | 安于现状 | 缓冲型 | P×0.7 | 野心事件权重-, 满足度+ | ambitious |

## 6. 执念生成逻辑

执念不是预设列表，而是基于**身份 + 特质组合**实时涌现的终极动机。

### 6.1 生成公式

```
执念 = f(身份, 特质1, 特质2, 特质3, 当前状态)
```

### 6.2 示例组合

| 身份 | 特质组合 | 涌现执念 |
|------|----------|----------|
| 教徒 | [狂热] + [精神错乱] | [成为上帝] |
| 教徒 | [慈悲] + [顾家] | [保护教友和家人] |
| 教徒 | [贪婪] + [狡诈] | [利用教门谋取私利] |
| 读书人 | [冷静] + [多疑] | [发现隐藏的真相] |
| 读书人 | [勇敢] + [狂热] | [为信仰传播而牺牲] |
| 读书人 | [怯懦] + [贪婪] | [用知识换取安全] |
| 官兵 | [愚忠] + [残忍] | [完成命令不择手段] |
| 官兵 | [怯懦] + [贪婪] | [搜刮后逃跑] |
| 官兵 | [公正] + [冷静] | [在乱世中维持秩序] |
| 地主 | [贪婪] + [野心勃勃] | [扩大势力成为土皇帝] |
| 地主 | [慈悲] + [顾家] | [散尽家财保护村民] |
| 地主 | [怯懦] + [多疑] | [怀疑所有人要夺产] |
| 流民 | [贪婪] + [狡诈] | [不择手段活下去] |
| 流民 | [勇敢] + [狂热] | [加入起义改变命运] |
| 流民 | [怯懦] + [疲惫] | [只想找到一口饭吃] |

### 6.3 状态影响

当前状态会修正执念表达：

| 状态 | 对执念的影响 |
|------|--------------|
| 极端恐惧 | [苟活下去]权重暴增 |
| 极端饥饿 | [获取食物]成为唯一执念 |
| 被背叛后 | 原执念 + [复仇]后缀 |
| 家人死亡 | 转为 [毁灭一切] 或 [自我毁灭] |
| 获得关键道具 | 执念与道具绑定 [保护/利用X] |

### 6.4 执念变化

执念不是固定的：
- **特质更新时重新计算** —— 核心机制
- 身份改变（如流民加入教门→教徒）重新生成
- 精神状态崩溃后执念扭曲

**特质-执念联动示例**：

| 原特质组合 | 原执念 | 事件 | 新特质 | 新执念 |
|------------|--------|------|--------|--------|
| [教徒]+[慈悲]+[顾家] | [保护教友和家人] | 家人被官兵杀害 | +[残忍]+[睚眦必报] | [血洗所有官兵] |
| [读书人]+[冷静]+[多疑] | [发现隐藏的真相] | 被信任者背叛 | +[偏执]+[狡诈] | [怀疑一切，只相信自己] |
| [官兵]+[愚忠]+[勇敢] | [效忠朝廷] | 上级出卖自己 | +[愤世嫉俗]+[怯懦] | [活下去，不再相信任何人] |
| [地主]+[贪婪]+[野心勃勃] | [扩大势力] | 财产被抄没 | +[绝望]+[狂热] | [毁灭这个世界] |

**联动规则**：
- 获得激化型特质（[残忍]/[狂热]/[睚眦必报]）→ 执念转向极端
- 获得缓冲型特质（[慈悲]/[冷静]）→ 执念趋于温和
- 获得[多疑]/[偏执] → 执念加入怀疑/验证元素
- 状态特质（[被背叛]/[饥饿]/[恐惧]）→ 执念与生存绑定

## 10. 角色生成规则

### 10.1 初始角色生成

1. 选择身份（决定B值和可选身份特质）
2. 随机抽取2-3个性格特质（检查互斥）
3. 基于特质组合生成执念
4. 随机初始状态值
