# 72Hours 玩家角色库 (Player Characters)

> 四个可选起始身份
> 每个身份预设基础属性，特质随机抽取

---

## 角色生成规则

```
1. 选择身份（决定B值和可选身份特质）
2. 从特质库随机抽取2-3个性格特质（检查互斥）
3. 基于身份+特质组合涌现执念
4. 随机初始状态值
5. 分配初始道具（基于身份和特质）
```

## 一、村中的读书人

### 基础属性

| 属性 | 数值 |
|------|------|
| 基础质量 B | 3 |
| 身份特质 | [读书人] |

### 可选性格特质倾向
- [冷静]、[多疑]、[狡诈]、[诚实]、[谦卑]、[傲慢]

### 身份效果
- [读书人]：P_traits × 0.8
- 信息获取能力增强
- 冲突中处于劣势
- 容易获得他人尊重

### 初始状态
- fear: 30
- aggression: 20
- hunger: 40

### 初始道具
- [线装书]（O=2，标签[知识][可焚毁]）

### 涌现执念示例

| 特质组合 | 涌现执念 |
|----------|----------|
| [读书人]+[冷静]+[多疑] | [发现隐藏的真相] |
| [读书人]+[勇敢]+[狂热] | [为信仰传播而牺牲] |
| [读书人]+[怯懦]+[贪婪] | [用知识换取安全] |
| [读书人]+[狡诈]+[野心勃勃] | [利用知识谋取私利] |

### 叙事特征
- 以智取胜，而非武力
- 容易获取情报和信任
- 高压下通过智慧降低冲突
- 物理弱势，易被威胁

### 示例1：读书人

