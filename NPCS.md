# 72Hours NPC库 (NPC Library)

> 金田起义（1851年1月8日-11日）角色库
> 包含5个精英NPC（条件解锁）和10个普通NPC（随机刷新）

---

## 第一部分：精英NPC (Elite NPCs)

精英NPC基于历史人物，具有预设特质和解锁条件，解锁后永久存在于世界。

---

### 1. 洪秀全 (Hong Xiuquan)

**身份**：拜上帝会创始人、太平天国天王
**基础质量 B**：10（最高位格）

**历史背景**：
广东花县人，四次科举失利后接触基督教，自称上帝次子、耶稣之弟。
创立拜上帝会，以"天下有大灾，唯信奉上帝可免难"传教。

**特质**：
- [狂热]（宗教极端）
- [野心勃勃]（建国欲望）
- [傲慢]（神权自居）
- [精神错乱]（后期明显，早期隐含）

**涌现执念**：
[建立天国，成为人间上帝] / [消灭清妖，拯救世人]

**解锁条件**：
- 回合 ≥ 60（1月10日之后）
- 或 环境压强 > 75
- 或 会众集结度 > 60

**首次出现叙事**：
> 远处传来低沉的诵经声。一个身穿黄袍的身影在火光中显现，众人纷纷跪倒。
> "我乃上帝次子，下凡诛妖。"他的目光扫过你，"你，可是来投天国的？"

**特殊效果**：
- 出现后自动吸引所有[教徒]向原点移动
- 与玩家互动时，若玩家有[教徒]特质，Knot暴增
- 若玩家拒绝皈依，hostility锁定为100

---

### 2. 杨秀清 (Yang Xiuqing)

**身份**：拜上帝会实际领袖、东王
**基础质量 B**：9

**历史背景**：
广西桂平人，煤矿工人出身，紫荆山烧炭工首领。
冯云山被捕后，以"天父下凡"稳定教众，救出冯云山。
金田起义实际组织者和军事指挥。

**特质**：
- [狡诈]（政治手腕）
- [野心勃勃]（权力欲望）
- [冷静]（危机处理）
- [领袖气质]

**涌现执念**：
[掌握实权，架空天王] / [建立不朽功业]

**解锁条件**：
- 回合 ≥ 50
- 或 会众集结度 > 40
- 或 玩家与[教徒]NPC互动次数 > 5

**首次出现叙事**：
> 一个瘦削的男子站在人群边缘，目光如炭火般幽深。
> 他没有跪拜，只是微微颔首。你注意到，洪秀全说话时，他在听；洪秀全停顿时，众人却在看他。

**特殊效果**：
- 与洪秀全同时在场时，形成"双星系统"，玩家可能被甩出
- 玩家若表现出能力，他试图拉拢
- 玩家若表现出忠诚于洪秀全，他对玩家产生敌意

### 3. 冯云山 (Feng Yunshan)

**身份**：拜上帝会创始人之一、南王
**基础质量 B**：7

**历史背景**：
洪秀全表亲，拜上帝会实际创建者。
在紫荆山区艰苦传教，以私塾教师身份掩护，发展数千信徒。
被乡绅王作新抓捕入狱，后被救出。

**特质**：
- [虔诚]（真信仰）
- [勤勉]（实干精神）
- [合群]（亲民）
- [体弱]（狱中受损）

**涌现执念**：
[传播真道，拯救苍生] / [辅佐天王，建立天国]

**解锁条件**：
- 回合 ≥ 40
- 或 玩家有[教徒]特质且Knot > 3
- 或 玩家携带[圣经]/[传教册子]

**首次出现叙事**：
> 一个面色苍白的教书先生坐在油灯下，手指抚过一本破旧的册子。
> 他抬头看你，目光温和却疲惫："你也来寻道的吗？"

**特殊效果**：
- 最容易建立信任关系的精英NPC
- 给予玩家[传教册子]道具
- 若他在场，玩家与[教徒]NPC互动时P降低

### 4. 韦昌辉 (Wei Changhui)

**身份**：北王、金田起义天使投资人
**基础质量 B**：8

**历史背景**：
广西桂平金田村人，家境殷实的小地主，受乡绅排挤加入拜上帝会。
变卖家产资助起义，是最重要的财政支持者。
后在天京事变中屠杀杨秀清全家，被洪秀全处死。

**特质**：
- [贪婪]（对财富敏感）
- [睚眦必报]（记仇）
- [狡诈]（隐藏野心）
- [富有]

**涌现执念**：
[报复所有轻视我的人] / [掌握财富和权力]

**解锁条件**：
- 玩家携带[银两]/[地契]等财富道具
- 或 玩家有[地主]身份
- 或 回合 ≥ 45 且 压强 > 50

**首次出现叙事**：
> 一个穿着比普通会众体面的中年人站在角落，手指无意识地摩挲着玉扳指。
> 他看你的眼神带着评估——像是在看一件待价而沽的商品。

**特殊效果**：
- 玩家若有财富，他试图交易或掠夺
- 玩家若表现出对乡绅的敌意，他产生好感
- 后期可能因权力斗争与杨秀清冲突，玩家被迫选边

### 5. 石达开 (Shi Dakai)

**身份**：翼王、太平天国的"另类"
**基础质量 B**：8

**历史背景**：
广西贵县人，16岁被洪秀全邀请加入拜上帝会。
年轻、英俊、慷慨好施，被称为"石敢当"。
变卖家产，募集4000人参加起义。
天京事变后率部出走，最终在大渡河被俘处死。

**特质**：
- [勇敢]（少年英雄）
- [慷慨]（散财聚人）
- [公正]（理想主义）
- [年轻]

**涌现执念**：
[建立公平正义的新世界] / [保护追随者]

**解锁条件**：
- 玩家表现出[勇敢]行为且被传播
- 或 玩家救助过其他NPC
- 或 回合 ≥ 55 且 玩家Knot总和 > 10

**首次出现叙事**：
> 一个少年骑马穿过人群，所过之处人们自动让开道路。
> 他跳下马，动作利落。你注意到他的眼睛——清澈，却藏着与年龄不符的沉重。

**特殊效果**：
- 玩家若有[逃兵]/[流民]身份，他主动提供庇护
- 与玩家建立关系后，Knot增长极快
- 若玩家陷入绝境，他可能率部救援

## 第四部分：关联NPC (Bonded NPCs)

关联NPC基于玩家起始身份生成，与玩家有预设关系，开局即存在。

---

### 读书人关联NPC

#### 读书人的母亲

**身份**：村妇，玩家的母亲
**基础质量 B**：4
**初始K值**：5（深厚亲情）

**特质**：
- [顾家]（一切为了孩子）
- [慈悲]（对他人仍有善意）
- [体弱]（年老）

**涌现执念**：
[保护儿子平安] / [维持这个家]

**叙事作用**：
- 情感锚点，最容易建立信任的NPC
- 可能为玩家藏匿物品或信息
- **死亡触发质量坍缩**：玩家M变负，无法回到"家"坐标

**开场叙事**：
> 她把一碗热粥推到你面前，手指粗糙，是常年劳作留下的。
> "外面乱，"她说，"喝完这碗，别出门了。"
> 她不知道你在做什么，但她知道你心里有事儿。

> "你来了，"他说，"上帝保佑，我们等你很久了。"
> 他的手心温热，但你知道这温热可能把你带向哪里。

**身份**：玩家的亲兄长，反对入教
**基础质量 B**：3

**特质**：

## 关联NPC通用规则

### 生成规则
```
玩家选择身份后：
1. 生成该身份对应的2个关联NPC
2. 设置初始K值（2-5）
3. 放置于玩家附近坐标
4. 标记为"已解锁"和"永久存在"
```

### 特殊机制

| 机制 | 说明 |
|------|------|
| 初始羁绊 | 关联NPC与玩家有预设K值，更容易成为聚光灯 |
| 情感放大 | 关联NPC的死亡/背叛对玩家影响×2 |
| 执念绑定 | 玩家的执念往往与关联NPC的命运绑定 |
| 质量坍缩 | 关联NPC死亡时，玩家M可能变负（斥力） |

### 叙事作用

关联NPC让玩家的选择有"重量"：
- 不是为了抽象的理想，是为了具体的人
- 不是为了历史大势，是为了母亲/父亲/兄弟
- 这让72小时的每一刻都有个人意义

## 第五部分：NPC升格与遗忘机制

### 3.1 普通NPC升格为精英NPC

**核心原则**：精英不是天生的，是涌现出来的。普通NPC通过与玩家的深度互动，可以"升格"为影响玩家命运的精英NPC。

#### 升格触发条件（满足其一）

| 条件 | 说明 | 示例 |
|------|------|------|
| 关系羁绊 K > 阈值 | 深度纠缠 | K >= 8 |
| 共同经历重大事件 | 生死与共 | 共同经历 >= 3次 |
| 玩家重大牺牲 | 情感债务 | 玩家为其冒生命危险 |
| 持有关键道具 | 命运绑定 | 持有玩家给予的[传家宝]等 |
| 关键时刻救玩家 | 恩情逆转 | 从被救者变为救命恩人 |

#### 升格效果

| 属性 | 变化 |
|------|------|
| 基础质量 B | 临时提升（+3） |
| 质量公式 | 解锁 S（叙事权重）累积 |
| 存在时间 | 从临时变为永久 |
| 标签 | 获得 [命运纠缠者] |
| 执念 | 与玩家深度绑定 |

#### 升格叙事示例

**逃兵李三的升格**：

```
初始：逃兵李三，B=3，临时NPC

事件链：
- 第10回合：玩家给予食物，K+2
- 第15回合：玩家保护他不被官兵发现，K+3，获得[被庇护]记忆
- 第25回合：李三为玩家挡刀，S+1，K+3
- 第30回合：K=8，触发升格

升格瞬间：
> 那个总是躲在角落的逃兵，此刻站在你身前，刀尖还在滴血。
> 他的眼神变了——不再是闪躲，而是某种执拗的坚定。
> "你救过我三次，"他说，"这条命是你的了。"

新属性：
- B=6（临时提升）
- 执念：[保护恩人]（与玩家绑定）
- 永久存在，直到死亡
```

### 3.2 质量遗忘机制

**适用对象**：普通NPC、升格NPC
**免疫对象**：历史预设精英NPC（洪秀全、杨秀清等）

#### 遗忘触发条件

```
若升格NPC：
- 连续 X 回合无互动
- G_trap 衰减至阈值以下
- 玩家主动切断关系

则触发质量遗忘：
- B 逐渐恢复原始值
- 执念淡化
- 最终淡出叙事视野
```

#### 遗忘叙事

```
轻度遗忘：
> 你忽然想起，已经很久没有他的消息了...

中度遗忘：
> 那个曾经为你挡刀的人，如今只记得他是个"逃兵"。

深度遗忘：
> 你甚至想不起他的名字，只记得曾经有过这么一个人。
```

### 3.3 精英NPC的稳定性

**历史预设精英NPC免疫以下机制**：
- 质量遗忘
- 执念淡化
- 自动消失

**原因**：
- 洪秀全必须在第60+回合出现
- 杨秀清必须推动起义进程
- 他们代表历史大势，不是个人情感可以改变的

**但玩家可以影响**：
- 与他们的关系（K值）
- 他们出现时的状态
- 他们对你的态度

### 3.4 双向升格

不仅NPC可以升格，玩家在某些NPC眼中也会"升格"：

```
若玩家：
- 多次救助某个NPC
- 在关键时刻影响历史走向
- 积累足够的S和K

则在该NPC的视角中，玩家成为"高引力源"，
NPC的行为模式会向玩家偏移，形成双向纠缠。
```

### 3.5 精英NPC的双重来源

| 来源 | 代表 | 特点 | 质量稳定性 |
|------|------|------|------------|
| 历史预设 | 洪秀全、杨秀清 | 代表历史大势 | 免疫遗忘 |
| 涌现升格 | 李三、张氏 | 代表个人命运 | 可能遗忘 |

**设计目标**：
- 历史大势（洪秀全）不可避免
- 但谁与你同行，由你的选择决定
- 每个玩家的"精英NPC阵容"都是独特的

---

### 精英NPC生成

```
每回合检查：
  FOR each 未解锁精英NPC:
    IF 满足解锁条件:
      生成NPC（使用预设属性）
      放置于特定坐标或玩家附近
      触发首次遭遇事件
      标记为"已解锁"
```

### 普通NPC生成

```
每回合：
  随机决定是否刷新普通NPC（概率基于压强）
  IF 刷新:
    从10个模板中随机选择
    随机生成姓名、微调特质
    随机放置于网格边缘
    设置存在回合数（5-15回合后消失）
```

### 质量对比

| 类型 | 质量范围 | 存在时间 | 叙事作用 |
|------|----------|----------|----------|
| 精英NPC | 7-10 | 永久 | 推动剧情，核心冲突 |
| 普通NPC | 1-4 | 临时 | 填充世界，偶然碰撞 |
