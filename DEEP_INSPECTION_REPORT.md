## 三、潜在 Bug

### 3.1 异步处理

**问题：** `server.js` 中 `await game.executeTurn()` 没有 try-catch

**位置：**
```javascript
// server.js:219
result = await game.executeTurn(choice);
```

**风险：** 如果 executeTurn 抛出异常，会导致服务器 500 错误

**状态：** 🟢 已在外层 try-catch 中处理

### 3.2 内存泄漏风险

**问题：** games Map 只增不减

**当前清理机制：**
- 每 30 分钟清理一次
- 过期时间 2 小时

**风险：** 如果短时间内创建大量游戏，内存可能耗尽

**建议：** 添加最大游戏数限制

### 3.3 没有输入消毒

**问题：** 用户输入直接传递到 AI 接口

**风险：** Prompt 注入攻击

**建议：** 添加输入验证和消毒

---

## 四、性能问题

### 4.1 AI 调用没有缓存

**问题：** 相同场景重复调用 AI

**影响：** 浪费 API 配额，响应慢

**建议：** 添加简单缓存（LRU）

### 4.2 没有连接池

**问题：** 每次请求新建连接

**影响：** 高并发时性能下降

**建议：** 使用 axios 连接池
